第2章 Meta元素配置

@Configuration
public class OpenApiConfig {
    @Bean
    public OpenAPI customOpenAPI() {
        return new OpenAPI().info(new Info()
                .title("Angin's API")
                .contact(new Contact().name("wangzilin").email("wangzilinn@gmail.com").url("zilinn.wang"))
                .description("Angin的api们")
                .license(new License().name("Apache 2.0").url("https://www.apache.org/licenses/LICENSE-2.0.html"))
                .version("1.0")
        );
    }
}

第3章 常用注解

3.1 POJO中:

3.1.1 @Schema

POJO中主要使用@Schema注解来标识属性的提示信息

3.1.2 JSR-303注解支持

3.1.3 示例:

@Data
@NoArgsConstructor
public class User implements UserDetails {

    @Schema(description = "用户昵称")
    private String nickname;
    @Email
    @Schema(description = "用户Email", required = true)
    private String email;
    @NotEmpty
    @Schema(required = true)
    private String password;

    private String affiliationId;
    @Schema(description = "用户角色:\"admin\", \"user\", \"guest\"", example = "admin", defaultValue = "user")
    private String role;
    @Schema(defaultValue = "用户访问所有api须带的Token")
    private String token;

}
